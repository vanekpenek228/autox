Что такое CMS?

CMS - "система управления контентом"

Движок - написание PHP- программистами основа для сайта, с помощью которой вы сможете управлять сайтом (добовлять контент, менять пункты, меню и т.п.) не зная HTML и CSS

Однако для того чтобы сделать сайт с помощью CMS потребуются услуги и программистамиста и дизайнера, и верстальщика. И капиталовложения.

Какие бывают CMS

Бывают различные системы управления контентом

для интернет - магазинов, для блогов, для формул и т.д.
